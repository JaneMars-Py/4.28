# MyDict: Реализация собственного словаря

## Описание

`MyDict` — это Python класс, который реализует базовый функционал словаря, но без использования встроенного типа данных `dict`. Он хранит пары "ключ-значение" в виде списка кортежей и поддерживает основные операции, такие как добавление, удаление, поиск, а также методы для получения ключей и значений.

### Возможности:
- Добавление новых пар "ключ-значение"
- Получение значения по ключу
- Удаление элемента по ключу
- Проверка наличия ключа в словаре
- Получение списка ключей, значений и пар "ключ-значение"
- Удобное строковое представление словаря

## Инструкция по установке

Чтобы использовать этот класс, вам не нужно устанавливать никаких зависимостей. Просто скопируйте код в файл Python и используйте его в своём проекте.

1. Клонируйте или скачайте репозиторий.
2. Вставьте код в файл `.py` (например, `mydict.py`).
3. Импортируйте класс `MyDict` в ваш проект и создавайте экземпляры словаря.

Использование

python
Копировать код
# Создание экземпляра MyDict
my_dict = MyDict()

# Добавление элементов
my_dict['name'] = 'Alice'
my_dict['age'] = 30

# Получение элементов по ключу
print(my_dict['name'])  # 'Alice'
print(my_dict['age'])   # 30

# Проверка наличия ключа
print('city' in my_dict)  # False

# Удаление элемента
del my_dict['age']

# Получение всех ключей
print(my_dict.keys())  # ['name']

# Получение всех значений
print(my_dict.values())  # ['Alice']

# Печать словаря
print(my_dict)  # {name: Alice}
Методы класса
__init__(self)
Инициализирует пустой словарь.

__getitem__(key)
Возвращает значение по ключу. Если ключ не существует, возвращает None.

__setitem__(key, value)
Устанавливает значение по ключу. Если ключ уже существует, обновляет его значение, иначе добавляет новую пару "ключ-значение".

__delitem__(key)
Удаляет элемент по ключу. Если ключ не существует, ничего не делает.

keys()
Возвращает список всех ключей в словаре.

values()
Возвращает список всех значений в словаре.

items()
Возвращает список всех пар "ключ-значение" в словаре.

__contains__(key)
Проверяет наличие ключа в словаре (аналог оператора in).

__str__()
Возвращает строковое представление словаря для удобства отладки.
